# 🚗 BeamNG Mod Installer

**The easiest way to install BeamNG.drive mods - perfect for beginners!**

![BeamNG Mod Installer](https://img.shields.io/badge/BeamNG-Mod%20Installer-ff6b35?style=for-the-badge&logo=beamng&logoColor=white)

## ✨ Features

- 🔥 **Fire UI** - Sleek, modern, gamer-friendly interface
- 📋 **Copy & Paste** - Just paste any mod link and click install
- 🎯 **Beginner Friendly** - No technical knowledge needed
- 🚀 **Auto-Detect** - Automatically finds your BeamNG mods folder
- 📦 **Universal Support** - Works with most mod hosting sites
- ⚡ **Fast Installation** - Download, extract, and install in one click

## 🎮 Supported Sites

- **BeamNG.com** (Official Repository)
- **WorldOfMods**
- **Modland**
- **GitHub Releases**
- **Direct Download Links** (.zip files)

## 📥 Installation

### Option 1: Download Pre-built EXE (Recommended)

Once built, download the `.exe` installer from the `dist` folder and run it!

### Option 2: Build Yourself

```bash
# Install Node.js (if not already installed)
# Download from: https://nodejs.org/

# Navigate to the folder
cd beamng-mod-installer

# Install dependencies
npm install

# Build the Windows EXE installer
npm run build:win

# Your installer will be in the 'dist' folder!
```

## 🚀 How to Use

1. **Launch** BeamNG Mod Installer
2. **Copy** a mod URL from any supported site
3. **Paste** the link into the app (or click the Paste button!)
4. **Verify** the mods folder is correct (auto-detected by default)
5. **Click** INSTALL MOD
6. **Enjoy** your new mod in BeamNG.drive!

## 📁 Default Mods Folder Locations

### Windows
```
C:\Users\[YourName]\AppData\Local\BeamNG.drive\0.27\mods
```

### Manual Selection
If auto-detect fails, click "Browse" and navigate to your BeamNG installation's `mods` folder.

## 🛠️ Troubleshooting

### Mod folder not detected?
- Click "Browse" and manually select your BeamNG mods folder
- Make sure BeamNG.drive is installed

### Installation failed?
- Check that the URL is a direct download link
- Some sites may require manual download
- Make sure you have write permissions to the mods folder

### Mod not showing in game?
- Restart BeamNG.drive after installing
- Check if the mod is enabled in the mod manager
- Verify mod compatibility with your game version

## 📋 Requirements

- **OS:** Windows 10/11 (64-bit)
- **BeamNG.drive:** Version 0.27 or later
- **Node.js:** v18+ (for building only)

## 🎯 Tips for Noobs

1. **Always restart BeamNG** after installing new mods
2. **One mod at a time** - Install mods individually to avoid conflicts
3. **Check compatibility** - Make sure mods work with your game version
4. **Backup your saves** - Before installing many mods
5. **Mod manager** - Use BeamNG's built-in mod manager to enable/disable mods
